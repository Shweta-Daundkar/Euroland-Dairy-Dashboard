# Euroland Dairy Dashboard

Euroland Dairy Dashboard is a web-based system developed to manage dairy business operations. It helps manage vendors, employees, payments, enquiries, and user data through an easy dashboard interface.

## Features
- Admin and Employee Dashboard
- Vendor and Dealer Registration
- Client, Vendor, and Employee Payment Management
- Enquiry and Follow-up Management
- User Management System

## Technologies Used
- HTML
- CSS
- Bootstrap
- JavaScript
